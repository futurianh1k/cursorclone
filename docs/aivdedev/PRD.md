# Product Requirements Document (PRD)

## 1. 목적
금융권 VDE 환경에서 개발자의 생산성을 저해하지 않으면서,
보안 규제를 충족하는 AI 코딩 플랫폼을 구축한다.

## 2. 사용자
- 금융권 내부 개발자
- 외주/협력사 개발자(VDE 계정 기반)

## 3. 핵심 요구사항
### 기능
- 브라우저 기반 IDE
- AI 자동완성, 채팅, 코드 편집, 에이전트
### 비기능
- 완전 폐쇄망
- 사용자별 격리
- 중앙 감사 로그

## 4. 성공 지표
- AI 자동완성 수용률 ≥ 25%
- IDE 가동률 ≥ 99.9%
